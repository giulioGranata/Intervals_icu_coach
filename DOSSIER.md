### Session Details

| Session Type | Target Power     | Duration   | Purpose                         |
| ------------ | ---------------- | ---------- | ------------------------------- |
| VO₂max       | 233–264 W (Z5)   | 60–75 min  | Aerobic ceiling development     |
| Endurance    | 123–165 W (Z2)   | 90–180 min | Aerobic base, fat metabolism    |
| Sweetspot    | 185–213 W (SS)   | 60–90 min  | Threshold proximity, efficiency |
| Long Ride    | 123–175 W (Z2–3) | 3–4 h      | Volume, aerobic durability      |
| Recovery     | < 121 W (Z1)     | 30–45 min  | Active recovery, flush fatigue  |

### Recovery Protocol

**Recovery Triggers (Auto-Deload):**

- HRV ↓ > 20% → Rest or Z1 only
- RHR ↑ ≥ 5 bpm → Drop to Z1–Z2, no quality work
- Feel ≥ 4 → Reduce volume 40–50%, no intensity
- Two+ triggers → Full rest day

**Feel Scale:**

| Score | Meaning                                      |
| ----- | -------------------------------------------- |
| 1     | Excellent (fully recovered)                  |
| 2     | Good (normal fatigue)                        |
| 3     | Moderate (manageable tiredness)              |
| 4     | Fatigued (reduced readiness, deload trigger) |
| 5     | Exhausted (complete rest required)           |
